# **Hospital-Management-System-Php**
The main purpose of this project is to manage basic hospital activities like scheduling appointments by writing chief complaint, description of the cause and receiving prescription from doctor. 

There are two types of users viz. **Patient** and **Doctor**.

Refer Requirements.txt file for more details.

## Pre-requisites
* [Php](https://www.php.net/docs.php)
* [MySQL Database](https://www.mysql.com/)
* [Xampp](https://www.apachefriends.org/docs/)
* [Netbeans](https://netbeans.apache.org/)
