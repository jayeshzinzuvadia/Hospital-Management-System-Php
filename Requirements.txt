* Language - Php
* Database - MySQL

* List of Requirements implemented :-

R.1 : Login Functionality
        R.1.1 : System allows user to enter Username and Password for authentication
        R.1.2 : System authenticates the user and generates failure message for invalid credentials
        R.1.3 : System shows a link to logout if user has logged into the System

R.2 : Create New User Account
        R.2.1 : System allows a new user to fill up Registration Form and create it's new account

R.3 : Patient's Functionality
        R.3.1 : Create an appointment with a doctor by providing Chief Complaint and it's Description
        R.3.2 : Cancel a "pending" appointment with the doctor
        R.3.3 : View "confirmed" appointment with the doctor
        R.3.4 : Delete appointments "cancelled" by the doctor
        R.3.5 : View "checked" appointments of his/her past medical history

R.4 : Doctor's Functionality
        R.4.1 : Based on Patient's complaint, a Doctor can Schedule appointment with that patient by giving date and time.                                      Otherwise, he may Cancel the appointment request.
        R.4.2 : View his/her "confirmed" appointments with patients
        R.4.3 : View checked appointments of patients
        R.4.4 : Cancel Patient's appointment if a patient doesn't arrived on that mentioned appointment date and time 
        R.4.5 : Write Prescription to a patient's complaint
 
R.5 : View User Profile
        R.5.1 : System allows each User to View his own profile
        R.5.2 : System allows Users to Update "Profile Picture", "Phone Number" and "Address" fields
